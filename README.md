# PhotoNotes 1.0

<img src="https://s3-us-west-2.amazonaws.com/imagesgit/web_hi_res_512.png" width="100" height="100" />

*Why?* Take Photo of the teacher´s board, a classmate´s note, etc. You can doodle on them later.
